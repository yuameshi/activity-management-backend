# 智能校园活动管理系统前端页面与路由详细设计

---

## 项目架构

包管理：npm
打包工具：TypeScript + Vite
框架：React
状态管理：jotai
路由管理：React Router
UI 组件库：@mui/material
样式处理：CSS Modules
HTTP 请求：axios

## 源代码位置

`src/frontend/`

## 构建产物存放位置

public/

## 二、各页面用途与核心功能说明

### 1. 登录页 `/login`

-   用户输入账号密码登录系统
-   支持记住我、找回密码
-   登录后跳转首页

### 2. 注册页 `/register`

-   新用户注册，填写基本信息
-   注册成功自动登录或跳转登录页

### 3. 首页/仪表盘 `/`

-   活动推荐、我的活动快捷入口
-   统计数据概览（报名数、签到率等）
-   系统公告、最新消息

### 4. 活动列表 `/activities`

-   展示所有活动，支持筛选（类型、时间、状态）、搜索、排序
-   活动卡片点击进入详情

### 5. 活动详情 `/activities/:id`

-   展示活动全部信息、图片、报名状态
-   用户报名/取消报名、进入签到、提交反馈
-   管理员可编辑、审核、删除活动

### 6. 活动发布 `/activities/new`

-   用户/管理员填写活动信息，上传图片
-   支持保存草稿、提交审核

### 7. 活动编辑 `/activities/edit/:id`

-   编辑已发布或草稿状态的活动
-   修改活动信息、图片

### 8. 报名列表 `/activities/:id/registrations`

-   管理员查看某活动所有报名用户
-   支持导出报名数据

### 9. 签到页 `/activities/:id/attendance`

-   用户扫码签到或手动签到
-   查看签到状态、签到历史

### 10. 签到管理 `/activities/:id/attendance/manage`

-   管理员管理签到记录
-   设置签到状态（已签、迟到、未签），导出签到数据

### 11. 个人中心 `/profile`

-   展示与编辑个人信息
-   修改密码、头像、联系方式

### 12. 我的活动 `/profile/activities`

-   用户参与、发布、报名的活动列表
-   快捷进入活动详情、签到、反馈

### 13. 我的反馈 `/profile/feedbacks`

-   用户提交的所有反馈记录
-   可查看反馈详情、评分

### 14. 数据统计 `/statistics`

-   活动参与度、签到率、活跃度排行等数据可视化
-   支持筛选、导出统计报表

### 15. 消息通知 `/notifications`

-   展示系统消息、活动推送
-   支持未读提醒、已读标记

### 16. 反馈评价 `/activities/:id/feedback`

-   用户对活动的反馈与评分
-   管理员可查看所有反馈

### 17. 管理员后台 `/admin`

-   管理员总览，快捷入口至各管理模块

### 18. 用户管理 `/admin/users`

-   管理员管理所有用户
-   分配角色、禁用账号、重置密码

### 19. 活动审核 `/admin/activities`

-   管理员审核活动，管理所有活动状态
-   支持批量审核、删除

### 20. 报名/签到管理 `/admin/registrations`

-   管理员查看所有报名与签到记录
-   支持筛选、导出

### 21. 权限与系统设置 `/admin/settings`

-   管理员分配权限、配置系统参数
-   查看操作日志

## 三、路由设计原则

-   所有页面均采用 React Router 进行路由管理
-   支持嵌套路由与权限路由（如/admin 下仅管理员可访问）
-   路由与页面结构一一对应，便于维护与扩展
-   支持面包屑导航与动态路由参数（如活动 ID）
